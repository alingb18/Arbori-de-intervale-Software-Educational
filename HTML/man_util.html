Manual de utilizare - Generator
Pagina "Generator" are ca scop fixarea mai bună a informațiilor, oferind o imagine asupra structurii de date.

Mai jos sunt prezentate funcțiile de query și update, aplicate pe valori alese de utilizator.

Pentru a vedea animațiile trebuie completate 3 secțiuni.

Prima dintre ele este "Vector", iar sintaxa pentru acestă secțiune este următoarea, cu următoarele mențiuni:
1. Sintaxa: a[1],a[2],a[3],...,a[n]
2. n trebuie să fie mai mic sau egal cu 8
3. a[i] (1 ≤ i ≤ n) este al i-lea element din vector și poate lua orice valoare naturală între 1 și 9999999
4. caracterele se vor scrie unul lângă altul, fără spații între ele
5. exemplu de sintaxă: Vector = 1,45,29,38,10921,5 (se observă că n-ul nu trebuie transmis).

A doua secțiune este "Query", iar sintaxa pentru acestă secțiune este următoarea, cu următoarele mențiuni:
1. Sintaxa: L,R
2. (1 ≤ L ≤ R ≤ n)
3. caracterele se vor scrie unul lângă altul, fără spații între ele
4. exemplu de sintaxă: Query = 3,5 .

A treia secțiune este "Update", iar sintaxa pentru acestă secțiune este următoarea, cu următoarele mențiuni:
1. Sintaxa: poz,val
2. (1 ≤ poz ≤ n) (val este un număr între 1 și 9999999)
3. caracterele se vor scrie unul lângă altul, fără spații între ele
4. exemplu de sintaxă: Update = 4,3567 .
