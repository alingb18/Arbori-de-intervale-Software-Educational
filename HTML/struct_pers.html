Structura Arborilor de intervale persistenți
Arborii de intervale persistenți au în spate structura clasică de arbori de intervale. Diferența este dată de modul în care aceștia sunt stocați, anume prin pointeri. Acest fapt crează flexibilitate mai mare în implementare. Astfel, vom reține un struct cu 3 câmpuri: valoarea din nod, adresa fiului din stânga și adresa fiului din dreapta. Totodată, vom reține un vector de rădăcini, iar la fiecare nouă versiune, vom actualiza pointer-ul reținut la poziția anterioară din vector.
Cod - Structură 

        
struct node
{
    int val;
    node *l, *r;
    node(int x): val(x), l(NULL), r(NULL) {};
    node(node *ll, node* rr)
    {
        l=ll;
        r=rr;
        val=0;
        if(l) val+=l->val;
        if(r) val+=r->val;
    }
    node(node *c): val(c->val), l(c->l), r(c->r) {};
} *roots[NMAX];
