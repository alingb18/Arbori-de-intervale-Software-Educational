Funcția Build (Arbori de intervale persistenți)
Funcția Build funcționează aproape la fel ca în cazul structurii clasice, diferența fiind că de data aceasta returnează adresa nodului creat. Această mențiune implică faptul că adresa de memorie a nodului nu există în momentul apelului funcției, deci nu vom putea reține acest parametru ca în cazul arborilor de intervale clasici. Așadar, funcția va avea 2 parametri care definesc intervalul de acoperire al nodului actual și își va păstra toate proprietățile. Pentru simplitate, la crearea nodului am folosit un constructor definit în struct. Complexitate: O(n)
Cod 

        
node* build(int st, int dr)
{
    if(st==dr) return new node(0);
    int mij=(st+dr)/2;
    return new node(build(st, mij), build(mij+1, dr));
}
