Update
Funcția "update" (actualizare) are ca scop modificarea valorii unei poziții din vector și a tuturor nodurilor din arborele de intervale care conțin acea poziție în intervalul de acoperire.

Funcția va avea 5 parametri (nod, Lnod, Rnod, poz, val) (unde poz este poziția din vector, iar val este noua valoare).

Vom coborî în arborele de intervale până în frunza specifică poziției date, alegând la fiecare pas "deplasarea" către nodul care conține poziția în intervalul său.

Când ajungem în frunză modificăm informația, apoi similar funcției "build", valoarea fiecărui nod va fi combinația valorilor fiilor săi.

Astfel va fi parcurs doar lanțul dintre frunză și rădăcină care conține log2(N) noduri, aspect care dă și complexitatea funcției de O(log(N)).
Cod 


void update(int nod, int st, int dr, int poz, int val)
{
    if(st==dr)
    {
        aint[nod]=val;
        return;
    }
    int mij=(st+dr)/2;
    if(poz<=mij) update(2*nod, st, mij, poz, val);
    else update(2*nod+1, mij+1, dr, poz, val);
    aint[nod]=max(aint[2*nod], aint[2*nod+1]);
}
