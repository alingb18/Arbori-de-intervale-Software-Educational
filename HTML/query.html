Query
Funcția "query" (interogare) are ca rol găsirea răspunsului pentru un interval dat (vom nota capătul din stânga cu st, iar pe cel din dreapta cu dr).

Funcția va avea 5 parametri (nod, Lnod, Rnod, st, dr), iar aceasta parcurge arborele începând cu rădăcina și:

1. dacă este la un nod al cărui interval de acoperire este inclus complet în [st, dr], "adăugăm" la răspuns valorea din nodul respectiv

2. altfel, va continua parcurgerea în nodurile care au intersecția dintre intervalul de acoperire și [st, dr] nevidă.

Complexitatea acestei funcții este O(log(N)).
Cod 


int query(int nod, int l, int r, int st, int dr)
{
    if(l>=st && r<=dr) return aint[nod];
    int mij=(l+r)/2, minim=INF;
    if(st<=mij) minim=min(minim, query(2*nod, l, mij, st, dr));
    if(mij<dr) minim=min(minim, query(2*nod+1, mij+1, r, st, dr));
    return minim;
}
