Funcția Query (Arbori de intervale persistenți)
Funcția Query nu prezintă nicio diferență față de arborii de intervale clasici. Mai jos am prezentat o implementare pentru operația de sumă pe interval. Complexitate: O(log(N))
Cod 

        
int query(node* nod, int st, int dr, int qst, int qdr)
{
    if(st>=qst && dr<=qdr) return nod->val;
    int mij=(st+dr)/2, sum=0;
    if(qst<=mij) sum+=query(nod->l, st, mij, qst, qdr);
    if(mij<qdr) sum+=query(nod->r, mij+1, dr, qst, qdr);
    return sum;
}
