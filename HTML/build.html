Build
Funcția "build" are ca scop formarea unui arbore de intervale pe baza unui șir dat, (în problemă acest șir a fost notat cu A).

Pentru a parcurge arborele de intervale, avem nevoie de o funcție recursivă cu 3 parametri (nod, Lnod, Rnod).

Din fiecare apel pentru un nod care nu e frunză, vom face alte 2 apeluri către nodul din stânga și către cel din dreapta.

Utilizând funcția "build" vom parcurge arborele, iar când ajungem într-o frunză, punem în aint[nod] valoarea din vectorul A de la poziția Lnod (pentru că Lnod=Rnod). Apoi, pentru orice alt nod, valorea query-ului pentru intervalul pe care acesta îl acoperă va fi combinarea valorilor fiilor acestuia.

Complexitatea acestei funcții este O(N), deoarece parcurge fiecare nod din arborele de intervale o singură dată.
Cod 


void build(int nod, int st, int dr)
{
    if(st==dr) //nodul este frunză
    {
        aint[nod]=A[st];
        return;
    }
    int mij=(st+dr)/2;
    build(2*nod, st, mij);
    build(2*nod+1, mij+1, dr);
    aint[nod]=min(aint[2*nod], aint[2*nod+1]);
}
